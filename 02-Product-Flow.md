# CodeMRI – Product Flow

## User Journey

1. Login
2. Connect GitHub
3. Select Repository
4. Start Scan
5. Watch Live Progress
6. View Dashboard
7. Open Detailed Reports
8. Chat with Repository
9. Export PDF/Markdown

## Scan Flow

Repository
→ Clone
→ Parse
→ Dependency Graph
→ Metrics
→ AI Reasoning
→ Report
→ Dashboard

## AI Flow

Repository
→ Code Context
→ Structured Prompts
→ GPT Reasoning
→ Actionable Insights

## Dashboard Flow

Overview
→ Metrics
→ Hot Files
→ Architecture
→ AI Recommendations
→ Repository Chat
